- UNMAPPED_PRODUCT / IGNORED_PRODUCT / MISSING_ERP_PRODUCT_REFERENCE
- UNMAPPED_SUPPLIER / IGNORED_SUPPLIER / MISSING_ERP_SUPPLIER_REFERENCE
- INVALID_ACCESS_KEY / DUPLICATE_ACCESS_KEY
- INVALID_QUANTITY / UNIT_CONVERSION_REQUIRED
- MISSING_ITEM_COST / COST_COMPONENT_MISMATCH / INVOICE_TOTAL_MISMATCH
- XML_MISSING / XML_PARSE_ERROR / XML_ERP_*_MISMATCH
- TITLE_WITHOUT_INVOICE / INVALID_DUE_DATE / INVALID_OPEN_AMOUNT
- CROSS_STATION_DATA_LEAK
