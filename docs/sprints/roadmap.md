# Roadmap de desenvolvimento

Ordem obrigatória:

| Sprint | Tema |
|--------|------|
| 0 | Fundação (este repositório) |
| 1 | Usuários, postos, JWT |
| 2 | Cadastros mestres / mapeamento ERP |
| 3 | Central de cotações manual |
| 4 | Elegibilidade, custo equivalente, ranking |
| 5 | Conector XPERT + sync |
| 6 | Vendas e BI operacional |
| 7 | Compras, NF-e, contas a pagar |
| 8 | Compra vs melhor cotação |
| 9 | Índices externos |
| 10 | Variação, spread, repasse |
| 11 | Precificação, margem, dossiê |
| 12 | Dashboard executivo e produção |
| 13 | Automação de cotações (IA/portal) |

Cada sprint termina com: funcionalidade utilizável, migrations, testes, docs, Compose e relatório + script da próxima solicitação.
