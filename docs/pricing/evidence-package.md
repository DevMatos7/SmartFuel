# Pacote de evidências

Exportação usa **snapshots persistidos** (não recalcula).

Inclui: posto/produto, custo, cotações, margem, piso, cenários, preço recomendado/aprovado, impacto estimado, mercado, alertas, aprovações, anexos, implementação, auditoria, hash.

Anexos MinIO: `storage_key`, `sha256`, `content_type`, `size_bytes`, `original_filename` — sem sobrescrita.
