# Margem e markup

Todas as fórmulas usam `Decimal`.

```
gross_margin_per_liter = sale_price - comparable_cost
gross_margin_percentage = gross_margin_per_liter / sale_price   # se sale_price > 0
markup_percentage = gross_margin_per_liter / comparable_cost    # se cost > 0
```

**Não confundir margem com markup na UI.**

Trata-se de **margem bruta comercial estimada**, não lucro líquido.
