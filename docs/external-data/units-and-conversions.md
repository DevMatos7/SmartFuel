# Unidades e conversões

Unidades iniciais: `USD_PER_BARREL`, `BRL_PER_USD`, `BRL_PER_LITER`, `BRL_PER_CUBIC_METER`, `PERCENT`, `INDEX_POINTS`.

Conversões somente com regra explícita (ex.: m³ → L ÷ 1000).

`conversion_policy.forbid_auto_currency = true` impede conversão de moeda no conector.

Brent permanece em USD/barril; combinação com dólar é análise posterior (Sprint 10).
