# Qualidade de dados — índices externos

Códigos: `MISSING_VALUE`, `INVALID_DATE`, `INVALID_UNIT`, `INVALID_CURRENCY`, `DUPLICATE_OBSERVATION`, `CONFLICTING_OBSERVATION`, `OUT_OF_ORDER_OBSERVATION`, `UNEXPECTED_FREQUENCY`, `SOURCE_DELAYED`, `SOURCE_UNAVAILABLE`, `VALUE_OUTLIER`, `REVISION_DETECTED`, `PARSER_ERROR`, `AUTHENTICATION_ERROR`, `RATE_LIMITED`.

`VALUE_OUTLIER`: sinaliza variação percentual acima do limiar da série; **não exclui** o valor.
